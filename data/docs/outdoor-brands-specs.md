### Battery Cross-Compatibility

- 24V batteries ONLY work with 24V tools
- 40V tools use TWO 20V-class batteries in series (some overlap with 24V ecosystem)
- 60V batteries ONLY work with 60V tools
- 80V batteries ONLY work with 80V tools
- The systems are NOT cross-compatible between voltage families
- POWERALL 24V is cross-compatible with other brands in the POWERALL alliance

## Battery Compatibility Cross-Reference

### Critical Rule: Batteries Do NOT Cross Brands

There is NO cross-compatibility between different brands' battery platforms. A STIHL AK battery will NOT fit a Husqvarna tool. A Toro 60V battery will NOT work in a Greenworks 60V tool. Each brand's batteries are proprietary.

### Within-Brand Compatibility

| Brand | Cross-Compatible? | Notes |
|-------|-------------------|-------|
| STIHL | NO between AS/AK/AP | Three separate systems. AK batteries cannot power AP tools and vice versa. |
| Husqvarna | Partial | MAX (homeowner) and Pro batteries are different systems |
| Greenworks | NO between voltages | 24V, 40V, 60V, 80V are all separate platforms |
| Toro | YES | All 60V Flex-Force batteries work with all 60V tools |
| ECHO | YES | All eFORCE 56V batteries work with all eFORCE tools |
| WORX | YES (clever) | 20V batteries combine: 2x20V=40V, 4x20V=80V. One battery type powers everything |

### Best Single-Ecosystem Strategy

If you want to invest in ONE battery ecosystem for all outdoor power:

1. **Best overall (homeowner):** Toro 60V Flex-Force -- 75+ tools, one battery, great mowers and snow blowers
2. **Best budget:** Greenworks 60V -- excellent value, wide selection, aggressive pricing
3. **Best for tool variety:** WORX 20V PowerShare -- 75+ tools from one battery, but lower power
4. **Best for premium quality:** Husqvarna MAX -- fewer tools but exceptional build quality
5. **Best gas-to-battery transition:** ECHO eFORCE 56V -- same ECHO quality, huge Home Depot availability
6. **Best for professionals:** STIHL AP system -- dealer support, backpack battery for all-day runtime

## Gas-to-Battery Transition Guide

### State of the Transition (2026)

The outdoor power equipment industry is in active transition from gas to battery. Here is where each category stands:

| Category | Battery Ready? | Notes |
|----------|---------------|-------|
| Push Mowers | YES | Battery mowers match gas for yards under 1/2 acre |
| Self-Propelled Mowers | YES | Most brands offer competitive battery SP mowers |
| Zero-Turn Mowers | PARTIAL | Available but expensive ($4,000--$8,000) |
| String Trimmers | YES | Battery trimmers handle most residential needs |
| Handheld Blowers | YES | Battery blowers match or exceed gas handhelds |
| Backpack Blowers | PARTIAL | Battery versions exist but gas still dominates for all-day commercial use |
| Hedge Trimmers | YES | Battery hedge trimmers are now standard |
| Small Chainsaws (10"--14") | YES | Battery is excellent for pruning and light cutting |
| Medium Chainsaws (16"--20") | PARTIAL | Top-tier battery saws rival mid-range gas, but battery cost is high |
| Large Chainsaws (20"+) | NO | Gas still dominates for professional forestry |
| Snow Blowers | PARTIAL | Good for moderate snow; gas still wins for heavy, wet snow |
| Edgers | YES | Battery edgers are fully capable |
| Pressure Washers | PARTIAL | Battery/hybrid options emerging but gas dominates for heavy cleaning |

### When to Stay Gas

- You have more than 1 acre of heavy-duty cutting
- You are a professional arborist felling large trees daily
- You live in an area with 20+ inch snowfalls
- You need all-day runtime without battery swaps
- You are a commercial landscaper running a crew (though this is changing fast)

### When to Go Battery

- You have a standard suburban yard under 1/2 acre
- Noise restrictions matter (early morning use, noise ordinances)
- You want low maintenance (no gas, oil, spark plugs, winterizing)
- You value convenience (push button start, no pull cord)
- You want a single battery that powers multiple tools
